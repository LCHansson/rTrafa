# rTrafa: Access Swedish Transport Statistics from the 'Trafa' API

A pipe-friendly R client for the Trafa statistical API (api.trafa.se).
Provides a search-then-fetch workflow for discovering and downloading
Swedish transport and infrastructure data using a consistent
tibble-based interface.
